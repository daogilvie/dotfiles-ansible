# Dotfiles

This repo stores my dotfiles as Ansible roles (>2.4).
It's a WIP!

## Installing

`./install.sh OPTIONAL_TAGS_FOR_ROLES`

## Current Roles

| Role | Description                                                                                                        |
| ---- | ------------------------------------------------------------------------------------------------------------------ |
| fish | Installs fish shell, plus some config, then installs and executes [fisher](https://github.com/jorgebucaran/fisher) |
